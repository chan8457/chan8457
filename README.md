## About Me

저는 요리사를 꿈꾸며 특성화 고등학교 조리과를 졸업하고, 26세까지 **호텔·레스토랑·프랜차이즈 식당** 등 다양한 현장에서 일했습니다.  
요리사로 일할 때부터 “어떻게 하면 더 효율적일까”를 늘 고민했습니다.  
주문 순서를 최적화하고, 재고와 원가를 관리하며 가장 효율적인 루트를 찾는 일은 저의 본능이자 즐거움이었습니다.

하지만 코로나19로 외식업계가 급격히 위축되면서, 몸과 마음이 모두 지쳤고 결국 체중이 160kg까지 불어났습니다.  
그 위기를 계기로, 저는 제 삶의 ‘루트’를 근본부터 다시 설계하기로 결심했습니다.

그때 영화 *<머니볼>*과 게임 *<풋볼 매니저>*에서 저의 심장을 뛰게 하는 장면을 만났습니다.  
바로 **데이터로 남들이 보지 못하는 저평가된 가치를 증명하고, 팀을 승리로 이끄는 전략**이었습니다.  
게임 속에서 늘 즐겨 하던 일이 현실에서도 강력한 무기가 될 수 있다는 사실을 깨달았습니다.

목표가 생기자, 저는 먼저 **‘제 자신’부터 증명해보았습니다.**  
**160kg에서 85kg까지 감량**하며 제 인생의 가장 큰 ‘최적화’를 이뤄냈습니다.  
이 경험을 통해 저는 문제를 정의하고, 가설을 세우고, 실험으로 개선하는 과정을 몸으로 배웠습니다.

이후 **사이버대학교 빅데이터학과**에 진학했고,  
2025년 초에는 **데이터 분석 부트캠프(Codeit)**에서 협업 기반의 실제 데이터 프로젝트를 수행했습니다.  
그곳에서 팀원들과 함께 **데이터로 비즈니스 전략을 도출**하고,  
**머신러닝 모델의 수치를 개선**할 때 가장 큰 성취감을 느꼈습니다.  
과거 요리사 시절 ‘원가 절감’을 고민하던 저의 본능이 이제는 데이터 분석과 모델링으로 이어지고 있습니다.

저는 매일 성장하기 위해 배우고, 부족한 점을 개선하며 발전을 즐깁니다.

### 2. 채용 플랫폼 지원 퍼널 진단 및 단계별 개선 제안
이 프로젝트는 **채용 플랫폼의 전환율 하락 원인 진단 및 개선 제안**을 목표로 진행되었습니다.  
2022~2023년 MAU 하락이 두드러진 시점의 데이터를 분석하여,  
‘방문 → 공고 열람 → 이력서 작성 → 지원 완료’로 이어지는 **4단계 퍼널 전환율을 추적**했습니다.  

- **방문 → 공고 열람**: 경력 필터 중요도는 높지만 사용량 급감 → 세분화된 경력 구간 기반 **A/B TEST 제안**  
- **공고 열람 → 이력서 작성**: 생소한 스타트업 공고 다수로 신뢰 신호 부족 → RFM 기반 기업 세그먼트 후 **TOP 50/100 태그 노출 전략**  
- **이력서 작성 → 지원 완료**: 가장 큰 이탈 구간으로 확인 → **지원 현황 요약 박스 A/B TEST 제안**  

데이터 기반으로 각 퍼널 단계의 병목 원인을 도출하고, 구체적인 실험 가설로 개선 방향을 제시했습니다.

### 4. 서비스 종료된 10대 익명 투표 앱 데이터 분석
마지막 프로젝트는 **서비스 종료된 10대 익명 투표 앱의 로그 데이터**를 활용한 고급 분석 과제입니다.  
‘신입 데이터 분석가로서 신규 서비스 재출시 시나리오를 설계한다’는 가정 하에 진행했습니다.  

핵심 분석 축은 두 가지입니다:  
- **정착 실패 분석 (첫 24시간 내 행동 패턴)**  
- **이탈 예측 (7일 미복귀 예측 모델)**  

서비스는 피크 대비 DAU가 **97.7% 급락(316,872 → 7,429)** 했고,  
가입자 중 70%는 핵심 기능(질문/투표)을 경험하지 못했으며, 첫 주 이탈률은 44%에 달했습니다.  

이를 해결하기 위해  
- 첫 세션 내 ‘첫 투표 경험’ 유도 플로우 설계  
- 중심학생 네트워크(Influencer Node) 유지 전략  
- LightGBM 기반 **조기 이탈 경보 시스템**을 구축했습니다.  

또한 Airflow → GCS → BigQuery(Bronze/Silver/Gold) → BQML 학습 파이프라인을 구성하고,  
**대시보드 및 Discord 알림 자동화 시스템**까지 구현했습니다.
